# GitHub Grounds

This is my safe playground repo for learning GitHub.

> Rule: No secrets, private keys, client data, or proprietary Artisphere code go in this repository.

## What this repo is for
- Practicing GitHub basics.
- Testing simple files and experiments.
- Learning how repos, commits, and plugins work.

## Safe content examples
- HTML, CSS, and JavaScript practice files.
- Notes in Markdown.
- Fake sample data only.

## Not for this repo
- API keys.
- `.env` files.
- Stripe credentials.
- Real customer or business data.
- Private Artisphere code or planning docs.
